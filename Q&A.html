Question-1: What are the different ways to select an element in the DOM?
There are a lot of ways. However, the most common uses are id, class, tag, and querySelector.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML: Gets/sets the HTML content inside an element, even tags
innerText: Gets/sets the visible text, CSS and triggers layout reflow
textContent: Gets/sets all text content, even the hidden ones
Question-3: What is event delegation in the DOM?
A technique to attach a single event listener to a parent element to handle events triggered by its child elements.
Question-4: What is event bubbling in the DOM?
A mechanism where an event starts at the target element and propagates/bubbles up through its ancestor elements.
Question-5: How do you create, add, and remove elements using JavaScript?
create using createElement(), add using append() with parent and remove by simply setting innerHTML to empty, like innerHTML = ""
